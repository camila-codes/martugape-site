# MARTUGA PE 🌊🐢
![Preview do projeto](preview.PNG)
Protótipo institucional desenvolvido para o projeto MARTUGA PE, iniciativa voltada à preservação das tartarugas marinhas e educação ambiental em Pernambuco.

## ✨ Objetivo

O projeto foi desenvolvido como uma proposta visual moderna para fortalecer:
- presença digital;
- conscientização ambiental;
- divulgação das ações do projeto;
- processo de associação e participação da comunidade.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Responsividade
- GitHub Pages

## 🎨 Funcionalidades

- Hero institucional
- Seção sobre o projeto
- Área de ações ambientais
- Formulário de associação
- Design responsivo
- Navegação intuitiva
